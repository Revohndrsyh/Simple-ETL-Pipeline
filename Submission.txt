> Install Dependency yang digunakan
pip install -r requirements.txt

> Menjalankan berkas
python main.py

> Menjalankan unit test
python -m unittest discover tests

> Menjalankan test coverage
coverage run -m unittest discover tests

> Melihat hasil report
coverage report -m
